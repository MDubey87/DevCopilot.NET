
# DevCopilot.NET — RAG‑Powered AI Application using GitHub Models

A modern **.NET 8** AI application that demonstrates:
- Retrieval‑Augmented Generation (RAG)
- Chat & Streaming Chat
- GitHub Models Integration
- Embeddings‑based Semantic Search
- Minimal API Architecture

## Features
- Chat and streaming chat
- RAG with local knowledge base
- Semantic search using embeddings
- Swagger UI for testing

## Setup
1. Install .NET 8 SDK
2. Configure GitHub Models Token using user-secrets
3. Run the API: `dotnet run`

## API Endpoints
### POST /api/chat
### POST /api/stream
### POST /api/ask

## Knowledge Base
Place .md and .txt files in `DevCopilot.Api/KnowledgeBase/`.
